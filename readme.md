# ⚡ PokeConnect

A web application built with **Vite**, **HTML5**, **CSS3**, and **TypeScript**. 

---

## Installation guide

This project was build with:
* **Node.js**: `22.x` (LTS). 
* **Package Manager**: `npm` (bundled with Node).
* **Editor**: [Visual Studio Code](https://code.visualstudio.com/) .

### Local development

Follow these steps to get your local development environment up and running:

* Install dependencies
```
npm install
```
* Launch a server
```
npm run dev
```

### Deployment

To create a production-ready bundle (minified and optimized), run:
```
npm run build
```
The output will be generated in the /dist folder, which can be hosted on platforms like Vercel, Netlify, or GitHub Pages.
